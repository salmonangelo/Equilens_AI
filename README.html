🚀 Quick Start

Prerequisites

  • Python 3.10+
  • Git and virtual environment support
  • Modern browser (Chrome, Firefox, Safari, Edge)
  • Optional: Google Gemini API key for LLM explanations

1. Clone & Setup

git clone https://github.com/salmonangelo/Equilens_AI.git
cd Equilens_AI

# Create virtual environment
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # macOS/Linux

# Install dependencies
pip install -r requirements.txt

2. Configure Environment

cp .env.example .env
# Edit .env with your settings
# Optional: Add GEMINI_API_KEY=your-key-here

3. Start Backend

python main.py
# Backend starts on http://localhost:8080

4. Start Frontend

cd frontend
python -m http.server 5000
# Open http://localhost:5000

5. Run Analysis

  • Upload a CSV dataset
  • Specify protected attributes (gender, race, age, etc.)
  • Click "Run Fairness Analysis"
  • View results with metrics and AI-generated insights

Using Sample Data? Try the included sample_test_data.csv for a quick demo.

📡 API Endpoints

Health Check

GET /api/health
# {"status": "healthy", "service": "EquiLens AI", ...}

Upload Dataset

POST /api/v1/upload

# Request:
curl -X POST http://localhost:8080/api/v1/upload \
  -F "file=@sample_data.csv"

# Response:
{
  "dataset_id": "a1b2c3d4",
  "filename": "sample_data.csv",
  "rows": 1000,
  "columns": 5,
  "column_names": ["gender", "age", "approved", "predicted", "income"],
  "preview": [...]
}

Run Fairness Analysis

POST /api/v1/analyze

# Request:
{
  "dataset_id": "a1b2c3d4",
  "protected_attributes": ["gender"],
  "target_column": "approved",
  "prediction_column": "predicted",
  "model_name": "loan_approval_v2"
}

# Response:
{
  "status": "success",
  "metrics": {
    "gender": {
      "disparate_impact_ratio": 0.72,
      "demographic_parity_difference": -0.15,
      "equal_opportunity_difference": -0.18,
      "fairness_risk_score": 0.85
    }
  },
  "overall_fair": false,
  "explanation": "AI-generated insights..."
}

💡 Usage Example (Python)

import requests

BASE_URL = "http://localhost:8080"

# 1. Upload CSV
with open("data.csv", "rb") as f:
    response = requests.post(
        f"{BASE_URL}/api/v1/upload",
        files={"file": f}
    )
    dataset_id = response.json()["dataset_id"]

# 2. Run analysis
analysis = {
    "dataset_id": dataset_id,
    "protected_attributes": ["gender"],
    "target_column": "approved",
    "prediction_column": "predicted",
    "model_name": "loan_approval_v2"
}

response = requests.post(
    f"{BASE_URL}/api/v1/analyze",
    json=analysis
)

result = response.json()
print(f"Fair: {result['overall_fair']}")
print(f"Risk Score: {result['metrics']['gender']['fairness_risk_score']}")

🔧 Troubleshooting

Issue Solution
Port 8080 already in use BACKEND_PORT=9000 python main.py
Module not found error pip install -r requirements.txt
Gemini API errors Leave GEMINI_API_KEY empty for local testing
Upload fails (CORS) Ensure frontend on localhost:5000 and backend on localhost:8080
CSV parse errors Check UTF-8 encoding and header row
Memory errors Max file size is 10 MB; split large datasets
